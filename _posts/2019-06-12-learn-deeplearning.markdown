## TensorFlow Hub

使用TensorFlow Hub需要TensorFlow的版本在1.7以上

安装TensorFlow Hub

```
pip install tensorflow-hub
```

文档：[https://tensorflow.google.cn/hub/](https://tensorflow.google.cn/hub/)

Github：[https://github.com/tensorflow/hub](https://github.com/tensorflow/hub)
